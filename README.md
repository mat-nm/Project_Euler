# Entraînement à la programmation competitive
Ce dépôt rassemble certaines de mes solutions aux problèmes du site [Project Euler](https://projecteuler.net/archives), écrites dans le cadre d'un **entraînement à la programmation compétitive**.

## Nature du code

Ces solutions ont été conçues pour résoudre des problèmes algorithmiques sous contrainte de temps. L'accent a donc été mis sur la **rapidité de conception** et l'**efficacité d'exécution**, plutôt que sur la pureté du code, ou l'optimalité des implémentations.

## Compilation/Exécution

La majorité des codes est écrite en C et compile correctement avec **gcc 15.2.0**:
```sh
gcc -O2 -g -Wall -std=gnu11 pb1.c -o pb
```

Le reste s'exécute avec **python 3**:
```sh
python3 pb20.py
```
